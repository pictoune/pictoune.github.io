    Transforma a Tua Navegação na Web em Aprendizagem de Línguas

    Potencia a tua experiência com Anki com a extensão de navegador AnkiLingoFlash.

    ✨ Experiencia o poder de AnkiLingoFlash:

    1. Instala a extensão do navegador
    2. Navega nos teus sites favoritos como de costume
    3. Clica com o botão direito em qualquer palavra ou frase que queiras aprender
    4. Recebe instantaneamente flashcards gerados por IA que podem ser adicionados ao Anki sem sair do navegador!

    Instalar Agora

    💡 Por que escolher AnkiLingoFlash?

    • Flashcards Enriquecidos: Cria flashcards completos com uma tradução direta, uma definição, um mnemônico e um guia de pronúncia.
    • Integração Perfeita com Anki: Gera e adiciona cartões ao Anki diretamente do teu navegador, sem precisar digitar e interromper a tua rotina diária.
    • Aprendizagem Personalizada: Cria conjuntos de flashcards personalizados de acordo com os teus interesses, sem precisar baixar baralhos de qualidade duvidosa.
    • Sem Distrações: Sem anúncios, apenas foco puro na tua jornada de aprendizagem de línguas.
    • Independência de Flashcards: Todos os flashcards criados com AnkiLingoFlash permanecem no teu baralho Anki, mesmo se deixares de usar a minha extensão.
    • Aprendizagem Acessível: Começa com um generoso teste gratuito, depois continua com um custo mínimo usando os teus próprios créditos de IA - eu não cobro nenhuma taxa adicional!

    Integra AnkiLingoFlash na tua rotina diária de navegação e vê o teu vocabulário crescer dia após dia. 🌱

    💸 Preços

    AnkiLingoFlash oferece um número limitado de flashcards gratuitos para começar. Depois disso, podes usar a tua própria chave API da OpenAI, que é muito económica (um dólar pode gerar milhares de flashcards).

    Usar a Tua Própria Chave API da OpenAI

    1. Cria uma Conta na OpenAI: Vai ao site do ChatGPT e regista-te para uma conta.
    2. Gera uma Chave API:
      • Uma vez que tenhas uma conta, navega para a seção de chaves API.
      • Clica em Criar nova chave secreta.
      • Escolhe um nome para a tua chave, como AnkiLingoFlash.
      • Clica em Criar chave secreta.
      • Anota a chave gerada, pois precisarás de inseri-la nas configurações da extensão AnkiLingoFlash.
    3. Adiciona Fundos à Tua Conta:
      • No painel de controlo da OpenAI, vai ao menu Configurações.
      • Navega para a seção Faturação.
      • Adiciona um mínimo de $5 USD (ou o equivalente na tua moeda local) à tua conta usando um cartão de crédito ou outros métodos de pagamento fornecidos no site.
    4. Usa a Chave API em AnkiLingoFlash:
      • Copia a tua chave API e cola-a nas configurações da extensão AnkiLingoFlash.
      • Depois de inserir a tua chave API, clica no botão de validação.
      • Uma validação bem-sucedida será indicada pela mudança da cor da borda do campo de texto para verde.
      • Nota: Só precisas de inserir e validar a tua chave API uma vez. AnkiLingoFlash irá armazená-la de forma segura para uso futuro.
