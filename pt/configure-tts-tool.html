    Configurando a Ferramenta de Texto para Fala

    Para adicionar guias de pronúncia aos seus flashcards, precisa de configurar a ferramenta de texto para fala para o seu idioma alvo. Siga estes passos:

    1. Gere o seu primeiro flashcard como descrito no passo anterior.

    2. No Anki, vá para Ferramentas > Gerir Tipos de Notas.

    3. Deve ver o tipo de flashcard correspondente ao idioma da palavra ou frase que selecionou. Por exemplo, se selecionou um termo em inglês americano, procure por AnkiLingoFlash_Inglês (US) [1 nota].

    4. Selecione o tipo de cartão apropriado e clique em Fichas….

    5. No editor de cartões:
      • Verifique o Modelo do Verso.
      • Clique em OK para fechar qualquer mensagem que apareça.
      • Clique em Add TTS.
    6. Configure o serviço de texto para fala:
      • Selecione o serviço que deseja usar. Por exemplo, para inglês, pode escolher o serviço local gratuito Youdao Dictionary.
      • Escolha a voz apropriada (por exemplo, English, American [default] para o inglês americano).
      • Clique em Save, depois em OK.
    7. Configure os campos TTS:
      • Para a opção Field, selecione Seleção.
      • Para o campo Language, selecione o idioma desejado para o texto para fala.
    8. Finalize a configuração:
      • Clique em Insert into Back Template.
      • Feche a mensagem que aparece clicando em OK.
      • No Modelo do Verso, encontre a linha [Guia de pronúncia ainda não instalado e/ou configurado. Para instalar, clique aqui. Para configurar, clique aqui.] e substitua-a pela linha que acabou de ser adicionada (deve conter a etiqueta TTS). Por exemplo, se escolheu a voz de inglês americano com o Youdao Dictionary, a linha deve ser semelhante a: {{tts en_US voices=AwesomeTTS:Selection}}. Note que esta nova linha sempre começa com {{tts.
      • Clique em Guardar.

    Importante: Terá de repetir este processo para cada idioma que deseja aprender.

    Parabéns! Agora completou totalmente a configuração do AnkiLingoFlash. Pode agora rever a primeira palavra ou frase que adicionou ao Anki, ou continuar a adicionar outros flashcards.
