    Konfiguration des Text-to-Speech-Tools

    Um Ausspracheführer zu Ihren Karteikarten hinzuzufügen, müssen Sie das Text-to-Speech-Tool für Ihre Zielsprache konfigurieren. Folgen Sie diesen Schritten:

    1. Generieren Sie Ihre erste Karteikarte wie im vorherigen Schritt beschrieben.

    2. Gehen Sie in Anki zu Extras > Notiztypen verwalten.

    3. Sie sollten den Karteikarten-Typ sehen, der der Sprache des von Ihnen ausgewählten Wortes oder Ausdrucks entspricht. Wenn Sie zum Beispiel einen amerikanisch-englischen Begriff ausgewählt haben, suchen Sie nach AnkiLingoFlash_Englisch (US) [1 Notiz].

    4. Wählen Sie den entsprechenden Kartentyp aus und klicken Sie auf Karten….

    5. Im Karteneditor:
      • Markieren Sie Vorlage für Rückseite.
      • Klicken Sie auf OK, um alle angezeigten Meldungen zu schließen.
      • Klicken Sie auf Add TTS.
    6. Konfigurieren Sie den Text-to-Speech-Dienst:
      • Wählen Sie den Dienst aus, den Sie verwenden möchten. Für Englisch können Sie beispielsweise den kostenlosen lokalen Dienst Youdao Dictionary wählen.
      • Wählen Sie die passende Stimme (z.B. English, American [default] für amerikanisches Englisch).
      • Klicken Sie auf Save, dann auf OK.
    7. Konfigurieren Sie die TTS-Felder:
      • Für die Option Field wählen Sie Auswahl.
      • Für das Feld Language wählen Sie die gewünschte Sprache für die Sprachsynthese.
    8. Schließen Sie die Konfiguration ab:
      • Klicken Sie auf Insert into Back Template.
      • Schließen Sie die angezeigte Meldung, indem Sie auf OK klicken.
      • In der Vorlage für die Rückseite finden Sie die Zeile [Aussprachehilfe noch nicht installiert und/oder konfiguriert. Zum Installieren klicken Sie hier. Zum Konfigurieren klicken Sie hier.] und ersetzen Sie sie durch die Zeile, die gerade hinzugefügt wurde (sie sollte den TTS-Tag enthalten). Wenn Sie beispielsweise die amerikanische englische Stimme mit Youdao Dictionary gewählt haben, sollte die Zeile so aussehen: {{tts en_US voices=AwesomeTTS:Selection}}. Beachten Sie, dass diese neue Zeile immer mit {{tts beginnt.
      • Klicken Sie auf Speichern.

    Wichtig: Sie müssen diesen Prozess für jede Sprache wiederholen, die Sie lernen möchten.

    Herzlichen Glückwunsch! Sie haben nun die gesamte Konfiguration von AnkiLingoFlash abgeschlossen. Sie können jetzt das erste Wort oder den ersten Ausdruck überprüfen, den Sie in Schritt 3 zu Anki hinzugefügt haben, oder fortfahren, weitere Karteikarten hinzuzufügen.
