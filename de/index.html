    Verwandeln Sie Ihr Surfen im Internet in Sprachenlernen

    Steigern Sie Ihre Anki-Erfahrung mit der AnkiLingoFlash-Browsererweiterung.

    ✨ Erleben Sie die Kraft von AnkiLingoFlash:

    1. Installieren Sie die Browsererweiterung
    2. Surfen Sie wie gewohnt auf Ihren Lieblingswebsites
    3. Klicken Sie mit der rechten Maustaste auf Wörter oder Ausdrücke, die Sie lernen möchten
    4. Erhalten Sie sofort KI-generierte Karteikarten, die direkt in Anki eingefügt werden können, ohne den Browser zu verlassen!

    💡 Warum AnkiLingoFlash wählen?

    • Erweiterte Karteikarten: Erstellen Sie umfassende Karteikarten mit einer direkten Übersetzung, einer Definition, einer Gedächtnisstütze und einer Ausspracheanleitung.
    • Nahtlose Anki-Integration: Erzeugen und fügen Sie Karten direkt von Ihrem Browser in Anki ein, ohne zu tippen und Ihren täglichen Ablauf zu unterbrechen.
    • Personalisiertes Lernen: Erstellen Sie personalisierte Karteikartensets, die auf Ihre Interessen zugeschnitten sind, ohne unsichere Qualitätsdecks herunterladen zu müssen.
    • Ablenkungsfrei: Keine Werbung, nur reine Konzentration auf Ihre Sprachlernreise.
    • Karteikarten-Unabhängigkeit: Alle mit AnkiLingoFlash erstellten Karteikarten bleiben in Ihrem Anki-Deck, auch wenn Sie meine Erweiterung nicht mehr verwenden.
    • Erschwingliches Lernen: Beginnen Sie mit einer großzügigen kostenlosen Testversion und setzen Sie dann zu minimalen Kosten fort, indem Sie Ihre eigenen KI-Credits verwenden - ich berechne keinen Aufschlag!

    Integrieren Sie AnkiLingoFlash in Ihre tägliche Surfroutine und sehen Sie, wie Ihr Wortschatz Tag für Tag wächst. 🌱

    💸 Preise

    AnkiLingoFlash bietet eine begrenzte Anzahl kostenloser Karteikarten, um Ihnen den Einstieg zu erleichtern. Danach können Sie Ihren eigenen OpenAI-API-Schlüssel verwenden, der sehr kostengünstig ist (ein Dollar kann Tausende von Karteikarten generieren).

    Verwendung Ihres eigenen OpenAI-API-Schlüssels

    1. Erstellen Sie ein OpenAI-Konto: Gehen Sie zur ChatGPT-Website und melden Sie sich für ein Konto an.
    2. Erzeugen Sie einen API-Schlüssel:
      • Sobald Sie ein Konto haben, navigieren Sie zur API-Schlüssel-Sektion.
      • Klicken Sie auf Neuen geheimen Schlüssel erstellen.
      • Wählen Sie einen Namen für Ihren Schlüssel, z.B. AnkiLingoFlash.
      • Klicken Sie auf Geheimen Schlüssel erstellen.
      • Notieren Sie den generierten Schlüssel, da Sie ihn in die Einstellungen der AnkiLingoFlash-Erweiterung eingeben müssen.
    3. Fügen Sie Ihrem Konto Guthaben hinzu:
      • Gehen Sie im OpenAI-Dashboard zum Menü Einstellungen.
      • Navigieren Sie zum Abschnitt Abrechnung.
      • Fügen Sie mindestens $5 USD (oder den entsprechenden Betrag in Ihrer Landeswährung) Ihrem Konto hinzu, indem Sie eine Kreditkarte oder andere auf der Website angebotene Zahlungsmethoden verwenden.
    4. Verwenden Sie den API-Schlüssel in AnkiLingoFlash:
      • Kopieren Sie Ihren API-Schlüssel und fügen Sie ihn in die Einstellungen der AnkiLingoFlash-Erweiterung ein.
      • Klicken Sie nach Eingabe Ihres API-Schlüssels auf die Schaltfläche zur Validierung.
      • Eine erfolgreiche Validierung wird durch das Ändern der Textfeldrahmenfarbe zu Grün angezeigt.
      • Hinweis: Sie müssen Ihren API-Schlüssel nur einmal eingeben und validieren. AnkiLingoFlash speichert ihn sicher für zukünftige Verwendungen.
