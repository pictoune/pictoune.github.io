    Trasforma La Tua Navigazione Web in Apprendimento delle Lingue

    Potenzia la tua esperienza Anki con l’estensione del browser AnkiLingoFlash.

    ✨ Scopri la potenza di AnkiLingoFlash:

    1. Installa l’estensione del browser
    2. Naviga sui tuoi siti web preferiti come al solito
    3. Fai clic destro su qualsiasi parola o frase che vuoi imparare

    💸 Prezzi

    AnkiLingoFlash offre un numero limitato di flashcard gratuite per iniziare. Dopodiché, puoi usare la tua chiave API di OpenAI, che è molto conveniente (un dollaro può generare migliaia di flashcard).

    Usare la Tua Chiave API di OpenAI

    1. Crea un Account OpenAI: Vai al sito web di ChatGPT e registrati per un account.
    2. Genera una Chiave API:
      • Una volta creato l’account, vai alla sezione chiavi API.
      • Clicca su Crea nuova chiave segreta.
      • Scegli un nome per la tua chiave, come AnkiLingoFlash.
      • Clicca su Crea chiave segreta.
      • Annota la chiave generata, poiché dovrai inserirla nelle impostazioni dell’estensione AnkiLingoFlash.
    3. Aggiungi Fondi al Tuo Account:
      • Nel dashboard di OpenAI, vai al menu Impostazioni.
      • Naviga alla sezione Fatturazione.
      • Aggiungi un minimo di $5 USD (o l’equivalente nella tua valuta locale) al tuo account utilizzando una carta di credito o altri metodi di pagamento forniti sul sito.
    4. Usa la Chiave API in AnkiLingoFlash:
      • Copia la tua chiave API e incollala nelle impostazioni dell’estensione AnkiLingoFlash.
      • Dopo aver inserito la tua chiave API, clicca sul pulsante di validazione.
      • Una validazione riuscita sarà indicata dal cambio di colore del bordo del campo di testo in verde.
      • Nota: Devi inserire e validare la tua chiave API una sola volta. AnkiLingoFlash la conserverà in modo sicuro per usi futuri.
