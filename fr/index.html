    💸 Tarification

    AnkiLingoFlash offre un nombre limité de flashcards gratuites pour commencer. Ensuite, vous pouvez utiliser votre propre clé API OpenAI, ce qui est très économique (un dollar peut générer des milliers de flashcards).

    Utilisation de Votre Propre Clé API OpenAI

    1. Créez un Compte OpenAI : Rendez-vous sur le site web de ChatGPT et inscrivez-vous pour un compte.
    2. Générez une Clé API :
      • Une fois que vous avez un compte, naviguez vers la section des clés API.
      • Cliquez sur Créer une nouvelle clé secrète.
      • Choisissez un nom pour votre clé, comme AnkiLingoFlash.
      • Cliquez sur Créer une clé secrète.
      • Notez la clé générée, car vous devrez l’entrer dans les paramètres de l’extension AnkiLingoFlash.
    3. Ajoutez des Fonds à Votre Compte :
      • Dans le tableau de bord OpenAI, allez dans le menu Paramètres.
      • Naviguez vers la section Facturation.
      • Ajoutez un minimum de 5 $ USD (ou l’équivalent dans votre devise locale) à votre compte en utilisant une carte de crédit ou d’autres méthodes de paiement proposées sur le site.
    4. Utilisez la Clé API dans AnkiLingoFlash :
      • Copiez votre clé API et collez-la dans les paramètres de l’extension AnkiLingoFlash.
      • Après avoir entré votre clé API, cliquez sur le bouton de validation.
      • Une validation réussie sera indiquée par le changement de la couleur de la bordure du champ de texte en vert.
      • Note : Vous n’avez besoin d’entrer et de valider votre clé API qu’une seule fois. AnkiLingoFlash la stockera en toute sécurité pour une utilisation future.
