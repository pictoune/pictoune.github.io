    Configuration de l’outil de synthèse vocale

    Pour ajouter des guides de prononciation à vos cartes mémoires, vous devez configurer l’outil de synthèse vocale pour votre langue cible. Suivez ces étapes :

    1. Générez votre première carte mémoire comme décrit dans l’étape précédente.

    2. Dans Anki, allez dans Outils > Gérer les types de notes.

    3. Vous devriez voir le type de flashcard correspondant à la langue du mot ou de la phrase que vous avez sélectionné. Par exemple, si vous avez sélectionné un terme en anglais américain, cherchez AnkiLingoFlash_Anglais (US) [1 note].

    4. Sélectionnez le type de carte approprié et cliquez sur Cartes….

    5. Dans l’éditeur de cartes :
      • Cochez Modèle du verso.
      • Cliquez sur OK pour fermer tout message qui apparaît.
      • Cliquez sur Add TTS.
    6. Configurez le service de synthèse vocale :
      • Sélectionnez le service que vous souhaitez utiliser. Par exemple, pour l’anglais, vous pouvez choisir le service local gratuit Youdao Dictionary.
      • Choisissez la voix appropriée (par exemple, English, American [default] pour l’anglais américain).
      • Cliquez sur Save, puis sur OK.
    7. Configurez les champs TTS :
      • Pour l’option Field, sélectionnez Sélection.
      • Pour le champ Language, sélectionnez la langue souhaitée pour la synthèse vocale.
    8. Finalisez la configuration :
      • Cliquez sur Insert into Back Template.
      • Fermez le message qui apparaît en cliquant sur OK.
      • Dans le Modèle du verso, trouvez la ligne [Guide de prononciation pas encore installé et/ou configuré. Pour l'installer, cliquez ici. Pour le configurer, cliquez ici.] et remplacez-la par la ligne qui vient d’être ajoutée (elle devrait contenir la balise TTS). Par exemple, si vous avez choisi la voix anglaise américaine avec Youdao Dictionary, la ligne devrait ressembler à : {{tts en_US voices=AwesomeTTS:Selection}}. Notez que cette nouvelle ligne commence toujours par {{tts.
      • Cliquez sur Enregistrer.

    Important : Vous devrez répéter ce processus pour chaque langue que vous souhaitez apprendre.

    Félicitations ! Vous avez maintenant entièrement terminé la configuration d’AnkiLingoFlash. Vous pouvez maintenant réviser le premier mot ou la première expression que vous avez ajouté(e) à Anki, ou continuer à ajouter d’autres cartes mémoires.
